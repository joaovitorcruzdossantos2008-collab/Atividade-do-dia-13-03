Validação de Senha com Exceção Personalizada em Java

Este projeto foi desenvolvido como atividade prática para reforçar conceitos de tratamento de exceções em Java e criação de exceções personalizadas.

O programa solicita que o usuário digite uma senha e verifica se ela atende aos critérios mínimos definidos pelo sistema.

Caso a senha não cumpra os requisitos (por exemplo, possuir menos de 8 caracteres), o sistema lança uma exceção personalizada chamada SenhaInvalidaException, que é capturada utilizando o bloco try/catch.

🧠 Conceitos aplicados

Tratamento de exceções (try/catch)

Criação de exceções personalizadas

Validação de dados de entrada

Estrutura básica de aplicações em Java

Orientação a Objetos

⚙️ Funcionamento

O programa solicita ao usuário que digite uma senha.

A senha é validada conforme os critérios estabelecidos.

Caso seja inválida, a exceção SenhaInvalidaException é lançada.

O erro é tratado e uma mensagem informativa é exibida ao usuário.

🛠 Tecnologias utilizadas

Java

IntelliJ IDEA

Programação Orientada a Objetos (POO)

🚀 Objetivo

Praticar o uso de exceções personalizadas e melhorar a organização do código ao lidar com erros e validações no desenvolvimento em Java.
